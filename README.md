<h1> TasksApp </h1>
<p>Lista de Tarefas em react native<p> 
  
  ### 🎲 Rodando o Servidor

```bash
# Clone este repositório
$ git clone <https://github.com/yantenorio/TasksApp.git>

# Acesse a pasta do projeto no terminal/cmd
$ cd taskapp

# Instale as dependências
$ npm install

# Execute a aplicação 
$ node app.js

```
